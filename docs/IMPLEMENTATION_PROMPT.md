# Рабочий промпт Chess Vision v0.4

Создай Android-приложение-тренера, которое разделяет два способа понимания шахмат:

1. **Vision** мгновенно оценивает текущую геометрию без глубокого дерева: мобильность,
   взаимную защиту, слабые фигуры и поля, давление, форпосты, линии и изменение потенциала
   после последнего хода.
2. **Stockfish** независимо проверяет позицию глубоким расчётом и показывает числовую оценку,
   глубину и главный вариант. Никогда не смешивай эти показатели и не называй эвристику
   движковой оценкой.

Обычная доска должна оставаться чистой. Пользователь сам включает режим подсказок, после чего
видит слабые/сильные поля, недостаточно защищённые фигуры, синергию и потенциальные угрозы.

Обучение строится не из одного примера на тему, а из серии различных позиций:

- короткое объяснение паттерна;
- самостоятельная попытка без стрелок;
- включаемая подсказка;
- проверка конкретного хода;
- объяснение ошибки и переход к следующему варианту;
- сохранение прогресса по теме.

Выбор дебюта перед игрой необязателен. Каталог должен содержать семейства и варианты,
характерный план и статистику белых/ничьих/чёрных из открытой базы. После выбора партия
начинается из позиции варианта; без выбора — из начальной позиции.

Код должен разделять шахматное ядро, обучение, дебютные данные, сетевые источники и UI.
Все сетевые функции обязаны иметь кэш, тайм-аут и понятное офлайн-состояние. Нелегальные ходы,
оставляющие короля под шахом, запрещены. Stockfish и сторонние OEX-движки подключаются только
с соблюдением лицензий и без ложных обещаний пользователю.
