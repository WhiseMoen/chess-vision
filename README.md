# Chess Vision

Android-приложение, которое объясняет шахматную позицию через **поле зрения, мобильность,
давление, защиту и потенциал фигур**, а не через длинную выдачу вариантов движка.

## Уже работает

- интерактивная доска и ввод любой позиции в FEN;
- карта поля зрения выбранной фигуры;
- доступные псевдолегальные поля и ручное перемещение фигур;
- подсчёт атакующих и защищающих фигур;
- поиск фигур под угрозой, запертых и незащищённых фигур;
- оценка потенциала каждой фигуры от 0 до 100;
- сравнение общего влияния сторон и контроля центра;
- пять оформлений доски: классика, дерево, океан, ночь и аметист;
- визуальные слои «Вилки», «Связи», «Давление» и «Сила»;
- стрелки двойных атак и давления, пунктирные линии взаимной поддержки;
- цветовые кольца сильных и ограниченных фигур;
- компактная горизонтальная история сыгранных ходов;
- учебный центр с 28 авторскими позициями, включая 10 самостоятельно решаемых задач;
- автоматическое распознавание связок, сквозных ударов, рентгена, батарей, форпостов и пешечных цепей;
- необязательный выбор из 25 дебютов и вариантов либо старт из обычной позиции;
- клиент статистики мастеров и продолжений Lichess Opening Explorer;
- встроенный Vision Bot, оценивающий взятия, безопасность, мобильность и влияние без глубокого поиска;
- независимая оценка Stockfish Cloud с фактической глубиной и главным вариантом;
- оценка эффекта хода: изменение мобильности, связей и влияния;
- выключаемые подсказки для слабостей, силы, синергии и вилок;
- рокировка, взятие на проходе, превращение пешки и фильтрация шаха;
- легальные ссылки на три public-domain учебника Project Gutenberg;
- объяснения на русском языке и учебные позиции.

## Запуск в Android Studio

1. Откройте папку проекта через **File → Open**.
2. Дождитесь Gradle Sync.
3. Выберите эмулятор или подключённый Android-телефон.
4. Нажмите **Run app**.

Минимальная версия Android — 8.0 (API 26), compile/target SDK — 35.

## Идея анализа

Для каждой фигуры строятся две разные карты:

- **vision** — все поля, на которые фигура направляет силу, включая занятые своими фигурами;
- **moves** — поля, куда она прямо сейчас может переместиться.

На их основе рассчитываются мобильность, безопасность, влияние на центр и итоговый потенциал.
Так приложение различает, например, активного слона, запертого слона и слона, который пока
закрыт, но уже защищает важную пешку.

## Визуальные слои

- **Обзор** — поле зрения выбранной фигуры и непосредственные угрозы.
- **Вилки** — жёлтые стрелки от фигуры к нескольким ценным целям.
- **Связи** — синие пунктирные линии поддержки между своими фигурами.
- **Давление** — красные стрелки к недостаточно защищённым фигурам.
- **Сила** — зелёные кольца активных фигур и красные кольца фигур с низким потенциалом.

Слои разделены, чтобы доска оставалась читаемой даже в сложной позиции.

## Ближайшее развитие

Следующий слой анализа: пешечные цепи и слабые поля, рентген по линиям, перегрузка защитника,
маршруты улучшения худшей фигуры и короткая тактическая проверка кандидатов через Stockfish.
Stockfish должен подтверждать, что объяснение тактически корректно, но не заменять само объяснение.

## Движки

В `v0.4.0-debug` используется встроенный **Vision Bot**. Он намеренно не является заменой
Stockfish: выбирает ход по локальным признакам позиции и помогает тренировать понимание
активности фигур.

Stockfish распространяется по GPLv3. При включении его бинарника приложение обязано предоставить
пользователю точный соответствующий исходный код. Сторонние Android-движки обычно подключаются
через Open Exchange Format (OEX); интерфейс выбора такого движка запланирован отдельным модулем.

Исследовательские выводы и источники: [docs/RESEARCH.md](docs/RESEARCH.md).

Глубокая оценка запрашивается отдельно у Stockfish Cloud API Lichess. Она требует интернет
и работает только для позиций, уже находящихся в облачном кэше. Opening Explorer с июля 2026
требует авторизацию: без неё приложение честно показывает требование входа, а не выдумывает
проценты побед. Токен добавляется нажатием на строку статистики и хранится только в закрытых
данных Android-приложения.
